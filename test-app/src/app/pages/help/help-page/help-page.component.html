How do I use the Re-searcher?

First, you must access the use tab and then click the "Terms" tab. In this tab you must add the terms that you want to carry out your search. Then, go to the "Logical operators" tab to add the operators of your search. Finally, access the "Result" tab to view the configured search strings.

 How do I set the complex terms? Those with hyphen "-" or apostrophes "`"?

The software automatically configures these terms. So just enter them normally.

 I found a mistake or a have new idea, where can I report it?

Please, consider open a new issue in GitHub repo.
